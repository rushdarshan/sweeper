# Contributing to ScreenshotSweeper

Thank you for your interest in contributing! Here's how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/ScreenshotSweeper.git`
3. Create a feature branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit: `git commit -am 'Add your feature'`
6. Push to your fork: `git push origin feature/your-feature`
7. Submit a Pull Request

## Areas for Contribution

- **Testing:** Unit tests for services and helpers
- **UI/UX:** Styling improvements, dark mode support
- **Features:** Custom presets, statistics dashboard, internationalization
- **Documentation:** Improved guides, video tutorials
- **Localization:** Translate to other languages

## Code Style

- Follow C# naming conventions (PascalCase for public members)
- Use meaningful variable names
- Add XML documentation comments for public methods
- Keep methods focused and testable

## Testing

Before submitting a PR:
- Test all affected features manually
- Add unit tests for new functionality
- Ensure existing tests still pass

## Reporting Issues

- Use GitHub Issues for bug reports
- Include steps to reproduce
- Include Windows version and .NET version info
- Attach screenshots if relevant

## Questions?

Feel free to open an issue or start a discussion!
