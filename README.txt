ScreenshotSweeper

A lightweight Windows desktop application that monitors and manages
screenshots with intelligent auto-deletion and a clean, tabbed UI.

This installer will place ScreenshotSweeper in your Program Files and
create shortcuts. The application is licensed under the GNU GPL v3.

See the `LICENSE` file included with this installer for the full text
of the GNU General Public License, version 3.

After installation you can open the app from the Start Menu or the
desktop shortcut (if selected).

If the .NET runtime is not present, the installer will prompt to open
the .NET download page.
